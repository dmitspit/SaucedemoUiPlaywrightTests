# Playwright Features

Here is a comprehensive list of Playwright features with links to their documentation pages:

1. **Writing Tests**  
   Write tests using web-first assertions, page fixtures, and locators  
   [Docs: Writing Tests](https://playwright.dev/docs/writing-tests)

2. **Running Tests**  
   Run single test, multiple tests, headed mode, parallel execution, retries, and more  
   [Docs: Running Tests](https://playwright.dev/docs/running-tests)

3. **Test Configuration**  
   Configure browsers, projects, reporters, and more  
   [Docs: Test Configuration](https://playwright.dev/docs/test-configuration)

4. **Codegen**  
   Generate tests automatically by recording user actions  
   [Docs: Codegen](https://playwright.dev/docs/codegen-intro)

5. **Trace Viewer**  
   See a trace of your tests for debugging and analysis  
   [Docs: Trace Viewer](https://playwright.dev/docs/trace-viewer-intro)

6. **HTML Test Reports**  
   View detailed HTML reports of your test runs  
   [Docs: Test Reporters (HTML)](https://playwright.dev/docs/test-reporters#html-reporter)

7. **UI Mode**  
   Run tests with a visual UI for debugging and time travel  
   [Docs: Test UI Mode](https://playwright.dev/docs/test-ui-mode)

8. **Browser Support**  
   Test on Chromium, Firefox, and WebKit, including mobile emulation  
   [Docs: Browsers](https://playwright.dev/docs/browsers)

9. **VS Code Extension**  
   Run and debug Playwright tests directly in VS Code  
   [Docs: Getting Started with VS Code Extension](https://playwright.dev/docs/getting-started-vscode)

10. **System Requirements**  
    Supported OS and Node.js versions  
    [Docs: System Requirements](https://playwright.dev/docs/intro#system-requirements)

11. **Installation & Updates**  
    How to install and update Playwright and browsers  
    [Docs: Installation](https://playwright.dev/docs/intro#installation)  
    [Docs: Updating Playwright](https://playwright.dev/docs/intro#updating-playwright)

12. **Community & Learning Resources**  
    Training, videos, Stack Overflow, Discord, and more  
    [Playwright Training](https://learn.microsoft.com/en-us/training/modules/build-with-playwright/)  
    [Learn Videos](https://playwright.dev/community/learn-videos)  
    [Feature Videos](https://playwright.dev/community/feature-videos)  
    [Community](https://playwright.dev/community)

Each feature above is linked to its official documentation for further details.
